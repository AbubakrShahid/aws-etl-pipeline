AWS ETL Pipeline
================

Overview
--------

This repository will contain a small AWS-based ETL pipeline managed with Terraform and implemented in Python.

- Detailed infrastructure, ETL logic, and dataset instructions will be added in later phases.

As the project evolves through each phase, this README will be expanded with setup steps, architecture details, and run instructions.
